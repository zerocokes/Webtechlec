How to set up the website:

1. Download WAMP Server from  wampserver.com/en/ and change the defaults.
2. Once the green "W" turns green, open the folder on where you installed your WAMP Server.
3. Inside the folder, find the folder named "www" and open it.
4. Create a new folder and name it WebtekLec2018.
5. Open your browser and type the "localhost/WebtekLec2018".
